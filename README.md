# blooprint

Go REST API template.

## Prerequisites

- Go v1.22
- Docker

## Get started

1. Create .env file

```
cp .env.example .env
```

2. Start database

```
make db-up
```

3. Run server

```
make run
```

## Stack

- Chi
- MongoDB

## Features

- Database
- Unit testing
- Containerization
- GitHub Actions
- Linting
- Logging
- Live reloading
